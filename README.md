A simple Python + SQLite project to manage bank customer data. It supports adding, viewing, updating, searching, and deleting customer records through a console-based interface.

->Features

-Add new customer records

-View all customers

-Update and delete records

-Search by name or account number

-Stores data securely in SQLite

-> Tech Stack

-Language: Python

-Database: SQLite3

-IDE: VS Code

->Learning Highlights

-Implemented CRUD operations using Python

-Integrated SQLite for data storage and management

-Strengthened understanding of Python–database interaction
